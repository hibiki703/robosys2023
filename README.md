# plusコマンド

![test](http://github.com/hibiki703/robosys2023/actions/workflows/test.yml/badge.svg)

標準出力から読み込んだ数字を足す。


（例）seq 3 | ./plus

例のように打ち込むと　1 + 2 + 3 を計算し、答えを出力する（この例だと 6 と出力される）
